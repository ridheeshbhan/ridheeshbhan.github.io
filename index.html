CEO Dashboard - Google Sheets

The snip has some visuals used on our CEO's Dashboard. This is our one stop shop.

Our data originates from Streamline (VRMS), which lacks an external API connection, so we export it regularly for analysis. A single macro is used to cleanse and transform the data before refreshing the dashboards and visuals. Sales and Revenue - Power BI

One of our clients needed a Power BI dashboard integrated with their Sage ERP system to visualize Sales, Purchases, Stock, Nominal Ledger, and Net Revenue. Since Sage supported AAD-based Power BI integration, we connected the data and used Power Query for cleansing and transformation.

Key requirements included drill-downs for the Nominal Ledger (from Code to Department to Cost Centre) and dynamic filters that refreshed in real-time. As Sage didn’t directly flag invoiced transactions, we built custom logic using helper columns to isolate invoiced data only.

We also incorporated benchmarking by comparing their performance against similar organizations, using external data provided by the client in Excel.

Sales Dashboard - Excel

A client needed a low-cost solution to generate weekly Sales reports segmented by Sales Rep → Sales Region → Stock details, replacing a slow, formula-heavy setup. We built a dynamic Excel Pivot Table powered by a partially cleaned SQL view, with plans to expand into KPI tracking and visuals.

Interactive filters were added for time ranges (e.g., current month, year, last 3 years), along with a macro-enabled button that auto-filtered each Sales Rep’s data, generated individual files, saved them to the server, and emailed them directly—fully hardcoded to avoid license costs.

The client was impressed by Excel’s capabilities, and the project expanded as a result of the system’s speed, automation, and simplicity.
